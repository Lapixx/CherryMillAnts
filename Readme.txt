Put your executable in this folder
edit playgame.bat
 - change myBot.exe into the name of your executable
 - change the username and password to your solisid and a chosen password

You can also specify a number of games that you want to play by adding a final optional argument

Example for 3 games:
tcpclient\tcpclient b2ki.science.uu.nl 2081 MyBot.exe username password 3
